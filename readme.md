## Descripción
Este proyecto retoma las gráficas elaboradas en el Proyecto 2 y las convierte en visualizaciones interactivas. Cada gráfica ahora muestra, junto a la imagen, una tabla con los datos utilizados para crearla. Al hacer clic sobre un elemento de la gráfica (una barra, un punto, etc.), se resalta automáticamente su fila correspondiente en la tabla, y viceversa.

## Contenido del repositorio
- Archivos de datos utilizados para cada gráfica
- Plantillas usadas para generar las imágenes
- Imágenes de las gráficas
- Páginas web interactivas de cada gráfica
- Página principal que enlaza las 6 visualizaciones
